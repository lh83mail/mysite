# 启动项目
1. 安装hexo 
https://hexo.io/zh-cn/docs/

```bash
npm install -g hexo-cli
```
   
2. 安装项目.首先在根目录 `npm install`
然后进入`themes/suka` 目录执行npm安装

```bash
npm install
cd themes/suka
npm install
cd ../../
hexo server
```
